# College Companion App

A simple, no-framework web app for college students to:
- view today's timetable
- add and track assignment deadlines
- remove completed assignments
- generate an AI-style **3-day step-by-step college study plan** by course, topic, goal type (exam/assignment/revision), start date, and available time
- get built-in subject helper guidance for chemistry, botany, zoology, mathematics, physics, english, and biology

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
